# Animal Shelter Manager
Mateusz Pisera and Jakub Sobala
===========================================================================
## What is this project about:
A Python CLI tool to manage animals, adopters, and medical records.

## This project features:
- Animal intake system
- Adopter management
- Medical logs
- Data validation using Pydantic
- JSON file storage
- Unit tests using pytest

## To run the project use:
python cli.py

## Run tests using:
pytest
